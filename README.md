# Document-Processing-Using-Cognitive-Science

## Prerequisites:
1.  Install Zbar:

    ```bash
    sudo apt install libzbar0
    ```

1. Install Tesseract:

    ```bash
    sudo apt install tesseract-ocr
    ```

1. Install Python Environment using Pipenv from Pipfile
